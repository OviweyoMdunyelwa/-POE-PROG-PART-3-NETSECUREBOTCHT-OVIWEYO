NetSecureBotChat
Project Overview

NetSecureBotChat is a desktop-based cybersecurity awareness chatbot developed using C#, Windows Presentation Foundation (WPF), and the .NET 8 framework. The purpose of the application is to educate users about cybersecurity threats, online safety, and best security practices through an interactive and user-friendly chatbot experience. The project was developed over three assessment phases, with each phase introducing additional functionality and improvements in both usability and intelligence.

Project Objectives

The primary objective of NetSecureBotChat is to increase cybersecurity awareness by providing users with reliable information about common cyber threats and recommended security practices. The chatbot was designed to demonstrate object-oriented programming principles while incorporating modern graphical user interface design, user authentication, conversation management, and intelligent response generation.

Development Phases
Part 1

The first phase focused on establishing the foundation of the chatbot application. This included introducing users to cybersecurity awareness through welcome messages, voice greetings, ASCII art, and predefined cybersecurity advice. The objective was to familiarise users with the application while demonstrating the basic concepts of chatbot interaction.

Part 2

The second phase introduced a graphical user interface developed using Windows Presentation Foundation (WPF). Users were able to create secure passwords, navigate between application pages, and interact with the chatbot through an intuitive interface. This phase also introduced user authentication and improved navigation throughout the application.

Part 3

The final phase significantly enhanced the chatbot by incorporating intelligent conversation features. These include keyword recognition, context awareness, sentiment detection, personalised responses, random response generation, conversation history, saved chat sessions, and improved user interaction. The chatbot can recognise cybersecurity-related topics and provide appropriate educational responses while adapting to the user's emotional state during conversations.

Key Features

The application provides a secure password creation process with password confirmation and show/hide password functionality. Returning users are able to log in using their previously created credentials.

The chatbot recognises cybersecurity topics such as password security, phishing, scams, malware, ransomware, online privacy, public Wi-Fi safety, and general cybersecurity practices. Responses are generated dynamically using collections and randomisation techniques to ensure that conversations remain engaging and varied.

The chatbot maintains conversational context by remembering previously discussed topics and the user's preferred cybersecurity interests. It is also capable of detecting emotional cues such as confusion, frustration, curiosity, and concern, allowing responses to be adjusted accordingly.

Users may save previous conversations locally and access them through the application. A typing indicator and modern user interface further improve the overall user experience.

Technologies Used

The project was developed using C# as the programming language and Windows Presentation Foundation (WPF) for the graphical user interface. Microsoft Visual Studio 2022 was used as the integrated development environment, while the application targets the .NET 8 framework.

System Requirements

To run the application successfully, the following software is required:

Windows 10 or Windows 11
Microsoft Visual Studio 2022
.NET 8 SDK
Running the Application

Open the solution in Microsoft Visual Studio 2022 and allow any required packages to restore. Build the solution before running the project. New users should first create and confirm a secure password before proceeding to the chatbot interface. Returning users may log in using their previously saved password before continuing to the chatbot.

Educational Purpose

This application was developed for educational purposes as part of a cybersecurity awareness project. The chatbot demonstrates the practical implementation of object-oriented programming concepts, graphical user interface development, file handling, collections, exception handling, and intelligent conversational techniques within a cybersecurity context.

Version History

Part 2: Introduction of the graphical user interface, password creation, authentication, and improved navigation.

Part 3: Intelligent chatbot implementation featuring context awareness, sentiment detection, personalised responses, conversation management, saved chats, and enhanced cybersecurity education.
